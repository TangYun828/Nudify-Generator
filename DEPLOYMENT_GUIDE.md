## 🌐 Deployment Paths

### Path 1: Local Development with Docker Compose
```bash
cd Nudify-Generator
python setup_env_vars.py
cd ..
docker-compose up
```
**Use for**: Testing locally, development
**Time**: 5 minutes
**Guide**: [DOCKER_QUICK_START.md](DOCKER_QUICK_START.md)

---

### Path 2: RunPod Serverless (Recommended for Production)
```bash
python setup_runpod.py
# Follow the printed instructions...
docker push yourusername/nudify-backend:runpod
# Create Serverless Endpoint in RunPod Console
# Add environment variables
```
**Use for**: Production with bursty traffic (< 100 req/day)
**Cost**: ~$7.50/month (1000 images)
**Time**: 5 minutes
**Guide**: [RUNPOD_QUICK_START.md](RUNPOD_QUICK_START.md)

---

### Path 3: RunPod Pod (Always On)
**Use for**: Continuous high-traffic service
**Cost**: $657+/month for RTX 4090
**Setup**: Same as Serverless but Pod instead of Endpoint
**Guide**: [RUNPOD_DEPLOYMENT.md](RUNPOD_DEPLOYMENT.md) → Option 2

---

### Path 4: Cloud Platform (AWS, Render, Railway, Heroku)
```bash
# Build image
docker build -t nudify-backend:prod .

# Push to Docker registry
docker push yourusername/nudify-backend:prod

# Deploy on platform (each has different steps)
# Set environment variables in platform dashboard
# C2PA_PRIVATE_KEY_BASE64, C2PA_CERT_BASE64, etc.
```
**Use for**: Enterprise, custom domains, complex infrastructure
**Guide**: [DOCKER_DEPLOYMENT.md](DOCKER_DEPLOYMENT.md) → Cloud Deployment

## 🔐 Security Best Practices

1. **Never commit .env files** to git
   - `.gitignore` already protects them
   - Use `python setup_env_vars.py` to generate

2. **Use RunPod Secrets** for production
   ```bash
   runpod secret create C2PA_PRIVATE_KEY "base64string"
   ```

3. **Restrict API access** by domain/IP
   - Only allow from your frontend
   - Use JWT tokens for authentication (already implemented)

4. **Enable HTTPS** (automatic on RunPod/Cloud)
   - Verify certificate: `curl https://api.yourdomain.com`

5. **Rotate credentials** yearly
   - Regenerate C2PA certificate
   - Update environment variables
